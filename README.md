# Spend Tracker

A simple web app to track what you spend and what you bought.

## Features

- Add purchases with:
  - item name
  - category (includes Food)
  - amount
  - date
- View totals for:
  - today
  - this week
  - this month
  - this year
  - all-time total spend
- Filter purchase history by category
- Delete single expenses
- Clear all saved expenses
- Data is saved in your browser using Local Storage

## Run

Open `index.html` in your browser.

For VS Code Live Server users:

1. Right click `index.html`
2. Select **Open with Live Server**
